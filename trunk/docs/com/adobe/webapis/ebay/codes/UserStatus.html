Class UserStatusConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class UserStatus

Provides constants for the UserStatusCodeType simple type.

This class identifies the code types that eBay could use to indicate the current state or status of an eBay user account.

See also

http://developer.ebay.com/DevZone/SOAP/docs/WSDL/xsd/1/simpletype/UserStatusCodeType.htm
