Packagecom.adobe.webapis.ebay.codes
Classpublic final class Severity

Provides constants for the SeverityCodeType simple type.

This class identifies the code types that eBay could use to indicate the severity of an API error. A code indicates whether there is an API-level error or warning that needs to be communicated to the client.
