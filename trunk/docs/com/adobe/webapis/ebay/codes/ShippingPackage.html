Class ShippingPackageConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class ShippingPackage

Provides constants for the ShippingPackageCodeType simple type.

This class identifies the code types that eBay could use to indicate the nature of the package used to ship the item(s). Required for calculated shipping only.

See also

http://developer.ebay.com/DevZone/SOAP/docs/WSDL/xsd/1/simpletype/ShippingPackageCodeType.htm
