Class PhotoDisplayConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class PhotoDisplay

Provides constants for the PhotoDisplayCodeType simple type.

This class identifies the code types that eBay could use to indicate the type of image display used in a listing.
