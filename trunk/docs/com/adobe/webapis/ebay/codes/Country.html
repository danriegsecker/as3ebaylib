Class CountryConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class Country

Provides constants for the CountryCodeType simple type.

This class defines the standard 2-letter ISO 3166 county code set, plus a few additional country codes specific to eBay.

See also

http://developer.ebay.com/DevZone/SOAP/docs/WSDL/xsd/1/simpletype/CountryCodeType.htm
http://www.iso.ch/iso/en/prods-services/iso3166ma/02iso-3166-code-lists/list-en1.html
