Packagecom.adobe.webapis.ebay.codes
Classpublic final class InsuranceOption

Provides constants for the InsuranceOptionCodeType simple type.

This class identifies the code types that eBay could use to indicate the seller's requirements regarding whether the buyer pays for the shipping insurance.
