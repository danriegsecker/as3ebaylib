Class EBayResultParserMethods
 
Packagecom.adobe.webapis.ebay
Classpublic final class EBayResultParser

Class that contains internal helper functions used to parse the XML returned from the eBay servers to create the relevant objects. Only meant as helpers for the library; NOT meant for use by developers using the library.
