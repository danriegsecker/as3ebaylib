Packagecom.adobe.webapis.ebay.codes
Classpublic final class VATStatus

Provides constants for the VATStatusCodeType simple type.

This class identifies the code types that eBay could use to indicate whether or not the user is subject to VAT. Users who have registered with eBay are VAT-exempt are not subject ot VAT.
