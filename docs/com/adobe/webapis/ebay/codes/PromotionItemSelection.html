Class PromotionItemSelectionConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class PromotionItemSelection

Provides constants for the PromotionItemSelectionCodeType simple type.

This class identifies the code types that eBay could use to indicate how the cross-promoted item was selected for association with the referring item. If the seller defined an item-to-item cross-promotion rule and the item was selected because of that rule, then "Manual" is returned.
