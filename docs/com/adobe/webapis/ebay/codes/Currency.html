Class CurrencyConstants
 
Packagecom.adobe.webapis.ebay.codes
Classpublic final class Currency

Provides constants for the CurrencyCodeType simple type.

This class defines the standard 3-letter ISO 4217 currency code set, plus a few additional currency codes specific to eBay.

See also

http://developer.ebay.com/DevZone/SOAP/docs/WSDL/xsd/1/simpletype/CurrencyCodeType.htm
http://www.xe.com/iso4217.htm
