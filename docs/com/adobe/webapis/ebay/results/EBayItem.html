Classpublic class EBayItem
InheritanceEBayItem EBayResult

Contains the results from a GetItem API call.

Extends EBayResult class, inheriting standard output fields.

Contains the following additional objects:

  • BestOfferDetails
  • BuyerRequirements
  • Category
  • Charity
  • ContactDetails
  • CrossPromotion
  • DigitalDeliveryDetails
  • EBayUser
  • ListingCheckoutRedirectPreference
  • ListingDesigner
  • ListingDetails
  • PaymentDetails
  • PictureDetails
  • ProductListingDetails
  • ReviseStatus
  • SellingStatus
  • ShippingDetails
  • StoreFront
  • VATDetails
