Packagecom.adobe.webapis.ebay.objects
Classpublic class Item

Contains summary data for the item listing itself, with the properties that define the listing.

Contains the following additional objects:

  • Category
  • DigitalDeliveryDetails
  • Distance
  • ListingDetails
  • PictureDetails
  • SearchDetails
  • SellingStatus
  • ShippingDetails
  • StoreFront
